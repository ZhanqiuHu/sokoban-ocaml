is_empty str returns true if str is empty or only contains spaces. Otherwise returns false.

is_invalid_go str_list is true only if str_list only contains one element: "go"

val is_malformed : string -> bool

is_malformed str is true if str is not in the form of "go <arg1> ... <argn>" or "quit" aftering removing spaces at the begining and at the end.

val form_go_command : string list -> Types.command

form_go_command str_list generates a command given str_list. Requires: str_list is a valid object phrase starting with go.

val parse : string -> Types.command

parse str parses a player's input into a command, as follows. The first word (i.e., consecutive sequence of non-space characters) of str becomes the verb. The rest of the words, if any, become the object phrase. Examples:

  • parse " w " is Go ["left"]
  • parse "quit" is Quit.

Requires: str contains only alphanumeric (A-Z, a-z, 0-9) and space characters (only ASCII character code 32; not tabs or newlines, etc.).

Raises: Empty if str is the empty string or contains only spaces.

Raises: Malformed if the command is malformed. A command is malformed if the verb is not one of "quit", "start", "w", "a", "s", "d"
