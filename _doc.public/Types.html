Module Types

module Types: sig .. end

types.mli contains all the types used for the game.

type player_num =
| Fst
| Snd

player_num is Fst if this is player one and Snd if this is player 2.

type player = {
   position : int * int;
   on_exit : bool;
   player_num : player_num;
   player_img : string;
}

player is the type for a player object.

type block = {
   position : int * int;
   in_hole : bool;
}

block is the type for a block that is pushable.

type hole = {
   position : int * int;
}

hole is the type for a hole.

type ttypes =
| Obstacle
| Normal
| Exit

ttypes is the type for any tile. It is either an obstacle (nonmovable block), exit, or normal walkable tile.

type breakable1 = {
   position : int * int;
   mutable hp : int;
}

breakable1 is the type for a breakable object.

type button = {
   mutable position : int * int;
   mutable width : int;
   mutable height : int;
   mutable image : string;
   mutable name : string;
   mutable enable : bool;
}

button is the type for a pushable button.

type select = {
   mutable position : int * int;
   mutable width : int;
   mutable height : int;
   mutable image_select : string;
   mutable image_deselect : string;
   mutable name : string;
   mutable enable : bool;
   mutable select : bool;
   mutable exclusives : select list;
}

select is the type for a selectable button. It can be selected or not selected.

type tile = {
   position : int * int;
   ttype : ttypes;
}

tile is the type for a single tile of the background.

type game_object =
| Player of player
| Block of block
| Hole of hole
| Break1 of breakable1
| Tile of tile
| Button of button
| Select of select

game_object is the type for a single object. It can be a player, block, hole, breakable, tile (background), button, or select.

type room = {
   room_id : string;
   width : int;
   height : int;
   mutable map_tile_list : tile list list;
   init_blocks : block list;
   holes : hole list;
   num_holes : int;
   exit_pos : int * int;
   init_pos : int * int;
   init_breaks : breakable1 list;
   step_limit : int;
}

room is the type for a single map of the game.

type modes =
| Normal
| Sliding
| Limit

modes is the type for a block mode of the game. It can be normal, sliding blocks, or step limit.

type state = {
   mutable mode : modes;
   mutable active : bool;
   mutable current_room_id : string;
   all_rooms : room list;
   mutable players : player list;
   mutable blocks : block list;
   mutable breaks : breakable1 list;
   filled_holes : int;
   mutable steps_left : int;
}

state is the type for a state of the game.

type history = {
   mutable state_list : state list;
   mutable num_steps : int;
}

history is the state history. It keeps track of previously visited states and the current number of steps.

type direction =
| Left
| Right
| Up
| Down
| Empty

direction is the type for a player movement. It can be left, right, up, down, or empty (none).

type command =
| Start
| Fst of direction
| Snd of direction
| Quit
| Back
| Pause
| Resume
| Return

command is the type for player input to the game. It can be start, player one movement, player two movement, quit, back, pause, resume, or return.
